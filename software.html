Scientific Tools

  • fourDvel

    Inference of time-dependent 3D velocity fields from displacement time series using Bayesian methods.

  • OffsetFields

    ISCE plugin for computing SAR and optical offset fields using cross-correlation and other methods.

  • PySAR

    Post-processing toolbox for synthetic aperture radar data, including InSAR, PolSAR, GPS, and seismic analysis.

  • Recon3D

    3D velocity reconstruction from multi-geometry InSAR observations with Laplacian smoothing priors.

  • VectorDisp

    3D displacement field estimation from InSAR line-of-sight measurements using multiple viewing geometries.

  • PolarDecomp

    Polarimetric SAR decomposition routines for target characterization and environmental monitoring.
